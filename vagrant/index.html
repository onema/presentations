Getting started with a dev environment

Created by Juan Manuel Torres / @onema

Requirements

  • GitHub Account
  • Git
  • Vagrant 1.3.4+
  • VirtualBox

https://github.com/onema/vagrant-lamp-development/blob/master/doc/tutorial.md#requirements

Notes go here!

Create SSH Key

     
$ cd ~/.ssh
$ ls         // Make sure not to overwrite an existing key pair
$ ssh-keygen -t rsa -C "your_email@example.com"

Clone Vagrant Lamp Dev Repo

     
$ cd ~/
$ mkdir lamp-development
$ cd lamp-development
$ git clone git@github.com:onema/vagrant-lamp-development.git vagrant
$ cp vagrant/Vagrantfile Vagrantfile

vagrant up --provision

Getting started with composer

"Composer is a tool for dependency management in PHP. It allows you to declare the dependent libraries your project needs and it will install them in your project for you."

Sample Code

In ~/Sites/project-name/composer.json


{
    "require": {
        "php": ">=5.3.3",
        "guzzle/guzzle": "3.7.*@dev"
    },
}

GET Google!

In ~/Sites/project-name/web/index.php

     
require_once __DIR__.'/../vendor/autoload.php';
use Guzzle\Http\Client;

$client = new Client('http://google.com');
$response = $client->get()->send();
echo $response->getBody(true);
